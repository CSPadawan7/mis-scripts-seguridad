# 🛡️ Mis Scripts de Seguridad

Repositorio donde documento mi camino de aprendizaje en **ciberseguridad** y **Python**, aplicado a casos reales de seguridad informática.

## Sobre mí

Soy ingeniero eléctrico con más de 12 años de experiencia, en transición hacia la ciberseguridad ofensiva (pentesting). Uso mi base técnica como ventaja para aprender redes, Linux y programación, conectando conceptos nuevos con analogías de ingeniería.

Cada script de este repositorio nace de un día de estudio estructurado, combinando teoría con práctica hands-on.

## Scripts

| Script | Qué hace | Conceptos practicados |
|--------|----------|----------------------|
| `analizar_ip.py` | Solicita nombre e IP y arma un mensaje de análisis | Variables, input, f-strings |
| `ip_detector.py` | Procesa información de IPs | Lógica condicional |
| `lista.py` | Recorre una lista de puertos | Listas, bucles for |
| `puertos.py` | Clasifica puertos por nivel de riesgo | Listas, condicionales, while |
| `funciones.py` | Verifica la fortaleza de una contraseña | Funciones, scoring, métodos de string |
| `funciones_while.py` | Práctica de funciones con bucles | Funciones, while |

## Stack

- **Lenguaje:** Python 3
- **Áreas:** Análisis de puertos, evaluación de contraseñas, hashing
- **En progreso:** Jr Penetration Tester path (TryHackMe)
